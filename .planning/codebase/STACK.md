## SPA Migration Note

The project migrated from **TanStack Start (SSR)** to a **pure Vite SPA**:
- `package.json` `name` is still legacy `"tanstack_start_ts"`.
- No SSR server entry; `src/main.tsx` does plain `createRoot().render()`.
- A stale `.tanstack/` directory remains from the Start era.
- Routing kept (TanStack Router file-based), but rendering is fully client-side.
- Deploy is static + SPA rewrite (see Platform Requirements).
